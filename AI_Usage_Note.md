# AI-Assisted Development

The development of this project was supported by multiple AI-powered tools to improve productivity, accelerate development, and enhance code quality.

## AI Tools Used

### Kiro

* Used for project architecture planning and code generation.
* Assisted in refactoring components and implementing new features.
* Helped generate boilerplate code and improve maintainability.

### Cursor AI

* Used during the initial development phase.
* Assisted with writing TypeScript, React, and Node.js code.
* Helped debug issues and suggest improvements during implementation.

### ChatGPT

* Used for problem-solving, debugging, and technical guidance.
* Assisted in designing the Linux hardening workflow.
* Helped generate documentation, test cases, project explanations, and presentation content.
* Provided recommendations for Docker integration, SSH troubleshooting, and Gemini AI fallback strategies.

## Role of AI in Development

The AI tools were used as development assistants to enhance efficiency and provide guidance. All architectural decisions, integration, testing, validation, and final implementation were reviewed and completed by the development team.

**Note:** AI tools supported the development process but did not independently build the complete application.
